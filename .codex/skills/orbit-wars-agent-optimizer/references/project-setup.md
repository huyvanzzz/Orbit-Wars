# Project Setup Standard

Create `AGENTS.md` in the repo root when missing. Keep it short and command-oriented.

Recommended sections:

- Scope
- Active Agent
- Read-Only Reference Files
- Benchmark Workflow
- Candidate Thresholds
- Replay Analysis
- Grid Search
- Notes

Include exact commands only when they exist in the repo. Prefer `.venv\Scripts\python.exe` on Windows when the project already has a `.venv`.

Do not claim a benchmark result unless it was actually run in the current session.
